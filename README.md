# PROJECT_TASK_TRACKER_CS
Console application for tracking your tasks

### How to Run Project
```bash
cd TaskTracker
dotnet run
```

### Menu
You have 5 points in menu:
- add (adding new task)
- list (showing all tasks)
- done (completing the task)
- wontdo (undoing the task)
- quit
